# VSCode Shortcuts

### 동시에 여러 줄 수정하기

```
cmd + option + 방향키
```

### 닫는 태그까지 한 번에 수정

```
cmd + d
```

### 주석

```
cmd + /
```

### 들여쓰기/내어쓰기

```
control + [     //들여쓰기
control + ]     //내어쓰기
```

### 탐색기 없애기

```
cmd + b
```

### 확장프로그램 [htmltagwap](https://marketplace.visualstudio.com/items?itemName=bradgashler.htmltagwrap)

- 자동으로 여는 태그 + 닫는 태그 만들어줌(기본적으로 p태그)

```
option + w
```
